## üöÄ Getting Started

### Installation

1. **Install Frontend Dependencies**
```bash
npm install
```

2. **Start the Application**
```bash
npm start
```

3. **Set up Backend** (in separate terminal)
```bash
cd backend
npm install
npm run dev
```

4. **Run AI Engine** (in separate terminal)
```bash
cd ai-engine
pip install -r requirements.txt
python app.py
```

Visit `http://localhost:3000` to see the magic! ‚ú®
